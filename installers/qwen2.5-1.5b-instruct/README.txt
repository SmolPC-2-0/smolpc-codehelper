SmolPC Code Helper — Qwen 2.5 1.5B Installer
==============================================

Lightweight model, recommended for 8 GB RAM machines.
Total bundle size: ~2.2 GB

Contents:
  - SmolPC Code Helper installer (~278 MB)
  - Qwen 2.5 1.5B DirectML model (~1.1 GB) — for GPU inference
  - Qwen 2.5 1.5B OpenVINO model (~777 MB) — for CPU/NPU inference

How to install:
  1. Double-click Install-CodeHelper.cmd
  2. Wait for the app and model to install (2-5 minutes)
  3. The app launches automatically

No internet connection required.
No admin rights required.

The app auto-detects your hardware and picks the best backend.
