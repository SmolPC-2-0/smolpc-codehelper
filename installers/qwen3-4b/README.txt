SmolPC Code Helper — Qwen 3 4B Installer
=========================================

Advanced model with better reasoning, recommended for 16 GB RAM machines.
Total bundle size: ~4.7 GB

Contents:
  - SmolPC Code Helper installer (~278 MB)
  - Qwen 3 4B DirectML model (~2.5 GB) — for GPU inference
  - Qwen 3 4B OpenVINO model (~1.9 GB) — for CPU/NPU inference

How to install:
  1. Double-click Install-CodeHelper.cmd
  2. Wait for the app and model to install (5-10 minutes)
  3. The app launches automatically

No internet connection required.
No admin rights required.

The app auto-detects your hardware and picks the best backend.
