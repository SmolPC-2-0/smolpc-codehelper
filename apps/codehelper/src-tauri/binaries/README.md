This directory is reserved for Tauri sidecar binaries.

Release automation copies `smolpc-engine-host` into this folder using
target-triple suffix naming required by Tauri:

- `smolpc-engine-host-x86_64-pc-windows-msvc.exe`
- `smolpc-engine-host-x86_64-unknown-linux-gnu`
- `smolpc-engine-host-aarch64-apple-darwin`
