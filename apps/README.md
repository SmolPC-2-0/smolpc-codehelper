# Apps Zone

Product applications that consume the shared SmolPC engine.

## Rule

Apps must use engine contracts (`smolpc-engine-client` or HTTP API), not engine host internals.

- `apps/codehelper`
- `apps/libreoffice-assistant` (skeleton)
- `apps/gimp-assistant` (skeleton)
